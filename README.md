# Sentinel AI

## Overview

Sentinel AI is an AI-powered Security Intelligence Platform designed to monitor and analyze security incidents across West Africa.

The first version focuses on:

- 🇬🇭 Ghana
- 🇨🇮 Côte d'Ivoire

## Features

- Automated incident monitoring
- AI-generated summaries
- Incident classification
- Risk prioritization
- Security dashboard
- Real-time alerts
- Daily intelligence reports
