# 🛡️ Sentinel AI

Sentinel AI is an AI-powered Security Intelligence Platform built with Python.

## Project Goal

The goal of Sentinel AI is to automatically monitor, collect, analyze, and store security incidents from trusted sources across West Africa, starting with Ghana and Côte d'Ivoire.

---

## Current Features

- ✅ RSS Feed Monitoring
- ✅ Security Keyword Filtering
- ✅ SQLite Database
- ✅ Automatic 5-Minute Scheduler
- ✅ Logging System
- ✅ Configuration File (JSON)

---

## Technologies Used
